# نظام إدارة التواصل الاجتماعي - iABS

## نظرة عامة

نظام متكامل لإدارة أعداد المتابعين لمنصات التواصل الاجتماعي الخاصة بـ iABS مع واجهة تحكم سهلة وقاعدة بيانات SQL كاملة.

## المميزات

### 🎯 الوظائف الرئيسية
- **تحديث يدوي لأعداد المتابعين** لجميع المنصات
- **سجل كامل للتحديثات** مع تواريخ وتفاصيل
- **نظام مصادقة آمن** للمديرين
- **واجهة عربية بالكامل** وسهلة الاستخدام
- **عرض إحصائيات مباشر** مع تنسيق الأرقام

### 📱 المنصات المدعومة
- **Instagram** - انستقرام
- **TikTok** - تيك توك  
- **X (Twitter)** - تويتر
- **WhatsApp** - واتساب
- **Snapchat** - سناب شات
- **Discord** - ديسكورد
- **YouTube** - يوتيوب
- **Kick** - كيك

## التثبيت والإعداد

### 1. قاعدة البيانات
```sql
-- استيراد مخطط قاعدة البيانات
mysql -u root -p iabs_social < database/social_media_schema.sql
```

### 2. إعدادات الاتصال
عدل ملفات API لتعديل بيانات الاتصال بقاعدة البيانات:
- `api/admin/social-stats.php`
- `api/admin/login.php`
- `api/admin/logout.php`
- `api/admin/check-auth.php`

```php
// تعديل هذه القيم
$host = 'localhost';
$dbname = 'iabs_social';
$username = 'root';
$password = '';
```

### 3. معلومات الدخول الافتراضية
- **اسم المستخدم**: `admin`
- **كلمة المرور**: `admin123`

## استخدام النظام

### 📱 الوصول إلى لوحة التحكم
1. افتح الرابط: `http://yoursite.com/pages/social-admin.html`
2. أدخل بيانات الدخول
3. اضغط على "دخول"

### ✏️ تحديث أعداد المتابعين
1. من لوحة التحكم، اختر "تحديث الأرقام"
2. اضغط على "تعديل العدد" بجانب أي منصة
3. أدخل العدد الجديد
4. اضغط "حفظ" لتأكيد التحديث

### 📊 عرض سجل التحديثات
1. اضغط على "سجل التحديثات"
2. عرض كامل التعديلات السابقة مع:
   - المنصة
   - العدد القديم
   - العدد الجديد
   - من قام بالتحديث
   - التاريخ والوقت

## هيكل قاعدة البيانات

### 📋 الجداول الرئيسية

#### `social_platforms`
- معلومات المنصات الاجتماعية
- الروابط والألوان والأيقونات

#### `social_media_stats`
- أعداد المتابعين الحالية
- تواريخ التحديث
- حالة التفعيل

#### `admin_users`
- بيانات المديرين
- الصلاحيات والمصادقة

#### `update_history`
- سجل كامل للتحديثات
- مصدر التحديث (يدوي/تلقائي)

## واجهة برمجة التطبيقات (API)

### 🔐 نقاط النهاية

#### `/api/admin/login.php`
- **POST**: تسجيل الدخول
- **المدخلات**: `username`, `password`

#### `/api/admin/logout.php`
- **POST**: تسجيل الخروج

#### `/api/admin/check-auth.php`
- **GET**: التحقق من الجلسة الحالية

#### `/api/admin/social-stats.php`
- **GET**: جلب الإحصائيات الحالية
- **POST**: تحديث عدد المتابعين
- **المدخلات**: `platform_key`, `follower_count`, `updated_by`, `notes`

### 📝 أمثلة الاستخدام

#### تحديث عدد المتابعين
```javascript
fetch('/api/admin/social-stats.php', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        platform_key: 'instagram',
        follower_count: 50000,
        updated_by: 'admin',
        notes: 'تحديث يدوي جديد'
    })
})
.then(response => response.json())
.then(data => console.log(data));
```

## استكشاف الأخطاء

### 🔧 المشاكل الشائعة

#### لا يمكن الاتصال بقاعدة البيانات
- تحقق من بيانات الاتصال في ملفات API
- تأكد من تشغيل MySQL/MariaDB
- تحقق من صلاحيات المستخدم

#### بيانات الدخول غير صحيحة
- تأكد من استخدام `admin` و `admin123`
- تحقق من جدول `admin_users` في قاعدة البيانات

#### لا يتم تحديث الأرقام
- تحقق من صلاحيات الكتابة في قاعدة البيانات
- راجع سجل أخطاء PHP
- تأكد من تفعيل JavaScript في المتصفح
